# ML_appliqu-_aux_donn-es_de_sant-
Ce projet analyse le jeu de données FAT pour prédire le taux de graisse corporelle (Brozek) via la régression linéaire multiple et pénalisée (Ridge, Lasso). Les travaux couvrent l'estimation de paramètres, la validation de modèles, la gestion de la multicolinéarité et la comparaison avec les outils R standards.
